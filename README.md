# Mini Curso - React js

Este repositório contém os materiais e informações relacionados ao mini curso **de React js**, ofertado ao Instituto Federal de Mato Grosso do Sul (IFMS) - Câmpus Naviraí, MS.

## Descrição

O mini curso de **React js** tem como objetivo introduzir os participantes às técnicas básicas de programação em react.

## Público-Alvo

Estudantes, professores e servidores do IFMS interessados em aprender a tcnologia react.

## Estrutura do Curso

O curso foi dividido nos seguintes tópicos:

1. **O que é** - O que é o React e como funciona;
2. **Estrutura de Pastas** - Como funciona a arquitetura de pastas do React;
3. **Primeiro Componente** - Estrutura de componentes e como funcionam;
4. **O que são PROPS** - Como deixar um componente dinâmico.

## Pré-Requisitos

- HTML básico;
- CSS básico;
- Javascript básico.

## Execução do Código

Para executar o código desenvolvido durante o curso, siga as etapas abaixo no terminal:

```bash
# Clone o repositório
git clone https://github.com/devPrates/cursoReact.git
```
```bash
# Navegue até o diretório do projeto
cd cursoReact
```
```bash
# Instale as dependências (caso aplicável)
npm install 
```
```bash
# Execute o código
npm run dev
```
